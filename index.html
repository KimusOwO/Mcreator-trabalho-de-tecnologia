Sobre o MCreator
Mcreator é um programa de criação de mods para o jogo minecraft, permitindo que você consiga fazer itens, blocos e até mesmo entidade sem saber programar. Ele possui uma programação parecida com a do Scratch, com programação por blocos, mas tambem a programação por linguagem java. Ele é disponível apenas para windows, porem os mods feitos podem ser direcionados para a versão JAVA ou Bedrock(versão compátivel com celular)
